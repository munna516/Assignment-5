Important FAQ

What is Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each element is a node in the tree.

How do you select the element from the DOM?

I can select the Element using getElementById(),getElementsByClass(), getElementsByTagName(), querySelector() function

What is event deligation in the context of DOM?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element to manage events for all of its child elements

How do You manipulate an elements attribute and styles using DOM?

You can use methods like getAttribute(), setAttribute(), removeAttribute(), and direct property manipulation to change an element's attributes.
